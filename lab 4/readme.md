## Description
This is a simple JavaScript quiz game that executes entirely within the browser console. It tests the user on general knowledge questions.

## How It Works
1. The game iterates through a predefined set of questions stored in an array.
2. It prompts the user for an answer.
3. User input is case-insensitive and ignores extra spaces.
4. Immediate feedback is provided via alerts.
5. The final score is displayed at the end.

## How to Run
1. Open a web browser (Chrome, Edge, Firefox).
2. Right-click the page and select **Inspect**.
3. Navigate to the **Console** tab.
4. Paste the code from `quiz.js` into the console.
5. Press **Enter** to start the quiz.
