# Lab 3: Responsive Personal Portfolio Website

## Description
This project is a one-page responsive portfolio website designed as part of the Web Dev I Lab Assignment 3. It utilizes advanced CSS techniques, including Flexbox, CSS Grid, Media Queries, and CSS Animations, to create a professional and interactive layout.

## Features
- **Responsive Design:** The layout adapts seamlessly to mobile, tablet, and desktop screens using a mobile-first approach.
- **Flexbox Layout:** Used for the header navigation and the hero section alignment.
- **CSS Grid:** Implemented for the structured layout of the Skills and Projects sections.
- **Animations & Effects:** Includes a color-changing text animation, fade-in effects upon loading, and hover transitions for buttons and cards.
- **Interactive Elements:** Smooth scrolling navigation and interactive form elements.

## Files Included
- index.html
- style.css
- README.txt

## How to Run
1. Download the `index.html` and `style.css` files to the same directory.
2. Open the `index.html` file in any modern web browser.

## Testing Responsive Behavior
To verify the responsiveness of the website, follow these steps:

1. Open the website in your browser.
2. Right-click anywhere on the page and select **Inspect**.
3. Toggle the **Device Toolbar** (or press Ctrl+Shift+M).
4. Test the layout at the following breakpoints:
   - **Mobile (approx. 375px):** The hero section text stacks vertically, and grid items in Skills and Projects appear in a single column.
   - **Tablet (approx. 768px):** The layout adjusts spacing, and grids display in two columns.
   - **Desktop (1024px and above):** The hero section displays side-by-side, and grids utilize the full width with multiple columns.
